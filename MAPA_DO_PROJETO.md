# MAPA DO PROJETO

## Estrutura

### `core/` (Fonte da Verdade)
| Arquivo | Função |
|---|---|
| `LP_VSL_COPY_FINAL.md` | Texto final da página (Copy) |
| `DESIGN_SYSTEM_NAIL.md` | Guia visual e tokens CSS |
| `README.md` | Guia de uso do Core |

### `guias/` (Guias de Implementação)
| Arquivo | Função |
|---|---|
| `GUIA_ESTRATEGIA_VISUAL.md` | Detalhamento de cada imagem/GIF (Prompts IA) |
| `GUIA_ESTRUTURA_BLOCOS.md` | Explicação da lógica narrativa da copy |

### `arquivo/` (Histórico Preservado)
- **_LEGADO_REDUNDANTE/**: Versões antigas ou duplicadas.
- **ASSETS/**: Imagens e mídias brutas.
- **03-PESQUISA/**: Pesquisa de público, VOC, cadernos NotebookLM.

### `/` (Raiz do Projeto)
- Código fonte (React/Next.js): `src/`, `public/`, `package.json`.
- Governança: `REGRAS_DO_WORKSPACE.md`, `CANONICAL_MAP.md`, `MAPA_DO_PROJETO.md`.

## Onde encontrar o que você precisa?

- **Texto para o site?** → `core/LP_VSL_COPY_FINAL.md`
- **Cor do botão?** → `core/DESIGN_SYSTEM_NAIL.md`
- **Gerar imagens com IA?** → `guias/GUIA_ESTRATEGIA_VISUAL.md`
- **Lógica da venda?** → `guias/GUIA_ESTRUTURA_BLOCOS.md`
- **Pesquisa de público?** → `arquivo/03-PESQUISA/`
- **Roteiro antigo?** → `arquivo/_LEGADO_REDUNDANTE/`
