## Regras

1. **Antes de criar qualquer arquivo**, consulte esta tabela.
2. Se o tema já tem canonical, **atualize o canonical** (append com data, se necessário).
3. Se o tema é novo, crie o canonical e **registre aqui imediatamente**.
4. Se encontrar 2+ arquivos do mesmo tema, **consolide em 1** e arquive os demais.

---

## Como adicionar tema novo

Adicione uma linha na tabela acima no formato:

```
| [Tema] | `[NOME_DO_ARQUIVO.md]` | `[PASTA/]` |
```

E crie o arquivo na pasta indicada.
