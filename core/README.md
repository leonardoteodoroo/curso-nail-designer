# 💎 DOCUMENTAÇÃO CORE - PROJETO NAIL DESIGNER

Este diretório contém os **ARQUIVOS MESTRES** inegociáveis do projeto.
Toda a execução da Landing Page deve derivar estritamente destas duas fontes de verdade.

---

## 1. 📝 COPYWRITING FINAL (`LP_VSL_COPY_FINAL.md`)

**O que é:**
O texto completo, aprovado e final da Landing Page VSL (Video Sales Letter em texto).

**Como usar:**
- **Copiar e Colar:** Não reescreva. O texto foi validado com técnicas de LightCopy e PNL.
- **Estrutura:** Siga a ordem dos blocos (Premissa -> Identificação -> História -> Produto).
- **Placeholders:** Onde houver `[PLACEHOLDER: Imagem X]`, consulte a pasta `/01-EXECUCAO/GUIA_ESTRATEGIA_VISUAL.md`.

**Nota Importante:**
Existe uma versão de análise (`PREVIEW_COPY_FINAL_ATUALIZADA.md` em Arquivados) que sugere uma reestruturação da oferta. Por regra de preservação, mantivemos este arquivo CORE intacto. Se desejar aplicar as mudanças do preview, consulte o arquivo na pasta de legado.

---

## 2. 🎨 DESIGN SYSTEM (`DESIGN_SYSTEM_NAIL.md`)

**O que é:**
A definição visual completa: Cores, Tipografia, Espaçamentos, Componentes e Motion.

**Como usar:**
- **Tokens:** Use as variáveis CSS definidas aqui (ex: `--action-primary: #E8B4B8`).
- **Look & Feel:** Respeite a estética "Rosa Suave + Dourado Premium".
- **Componentes:** Use as classes base (btn-primary, card, impact-box) para consistência.

---

**Regra de Ouro do CORE:**
Se precisar alterar algo fundamental (ex: mudar a cor da marca ou o preço do produto), **altere aqui primeiro**. Estes arquivos são a fonte da verdade.
